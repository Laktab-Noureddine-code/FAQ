# FAQ React App

This project is a simple FAQ (Frequently Asked Questions) page built with React. It allows users to toggle between a single question or multiple question selection to view the answers. The application is interactive and includes a button that enables multiple selection mode, allowing users to expand and collapse several questions at once.

## About the Project

This project demonstrates the use of React's state management (`useState`) to handle the toggling of questions and answers in a FAQ page. It includes the option to enable "multiple selection" mode, allowing users to open more than one question at a time.

### Features

- Toggle FAQ questions to show or hide answers.
- Enable or disable "multiple selection" mode.
- Each question can be expanded or collapsed.
- Responsive design to work across all devices.
